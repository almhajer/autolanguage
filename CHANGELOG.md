# سجل التغييرات

جميع التغييرات المهمة في امتداد "مبدل اللغة التلقائي" سيتم توثيقها في هذا الملف.

## [1.0.0] - 2024-02-07

### المطور
- عبدالكافي الحسن

### الميزات الجديدة
- إصدار أولي من امتداد مبدل اللغة التلقائي
- الكشف التلقائي عن النص العربي والإنجليزي
- التبديل التلقائي لتخطيط لوحة المفاتيح على Windows و macOS و Linux
- مؤشر شريط الحالة يعرض اللغة المكتشفة حالياً
- أمر التبديل لتمكين أو تعطيل الامتداد
- إعدادات قابلة للتخصيص للإشعارات وعرض شريط الحالة

### التحسينات
- الكشف البسيط للغة بناءً على عدد الأحرف في السطر
- التبديل فقط عند الانتقال لسطر جديد لتجنب التبديل المتكرر
- كود مبسط وفعال بدون أنظمة معقدة

### المنصات المدعومة
- ✅ macOS
- ✅ Windows
- ✅ Linux

### طرق التبديل
- **Windows**: PowerShell مع Win32 API
- **macOS**: AppleScript عبر osascript
- **Linux**: gsettings أو setxkbmap أو ibus أو fcitx

### الإعدادات
- `autolanguage.enabled`: تمكين أو تعطيل مبدل اللغة التلقائي
- `autolanguage.showNotifications`: عرض الإشعارات عند تبديل اللغة
- `autolanguage.showStatusBar`: عرض عنصر شريط الحالة مع اللغة الحالية
