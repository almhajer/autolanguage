# Auto Language Switcher

امتداد لـ Visual Studio Code يبدّل لغة لوحة المفاتيح تلقائيًا بين العربية والإنجليزية بناءً على النص القريب من موضع المؤشر.

![Auto Language Screen](resources/AutoLanguage-screen.gif)

## معلومات سريعة

- الاسم: `autolanguage`
- الإصدار الحالي: `0.0.6`
- المطور: عبدالكافي الحسن
- الأنظمة المدعومة: macOS / Windows / Linux

## الميزات

- كشف ذكي للغة من النص المحيط بالمؤشر.
- ترجيح أعلى للكلمة الموجودة عند موضع الكتابة لرفع الدقة.
- التعامل مع السطر الجديد الفارغ عبر الاستفادة من أقرب سياق نصي.
- عدم تنفيذ تبديل إذا كانت لغة الإدخال الحالية مطابقة بالفعل.
- مؤشر في شريط الحالة لعرض اللغة المكتشفة.
- إشعارات اختيارية عند تبديل اللغة.
- أمر سريع لتشغيل/إيقاف الامتداد.

## المتطلبات

- Visual Studio Code `1.60.0` أو أحدث.
- إضافة تخطيط لوحة مفاتيح عربي وإنجليزي في نظام التشغيل.

## التثبيت

### 1) التثبيت من ملف VSIX

1. حمّل الملف: `autolanguage-0.0.6.vsix`.
2. افتح VS Code.
3. افتح صفحة الامتدادات `Extensions`.
4. من قائمة `...` اختر `Install from VSIX...`.
5. اختر الملف `autolanguage-0.0.6.vsix`.

### 2) التثبيت من الطرفية

```bash
code --install-extension autolanguage-0.0.6.vsix
```

## طريقة الاستخدام

بعد التثبيت، الامتداد يعمل تلقائيًا:

1. حرّك المؤشر إلى نص عربي أو اكتب بالعربية في موضع معين.
2. سيحوّل الامتداد لغة الإدخال إلى العربية عند الحاجة.
3. حرّك المؤشر إلى نص إنجليزي أو اكتب بالإنجليزية.
4. سيحوّل الامتداد اللغة إلى الإنجليزية عند الحاجة.
5. إذا كانت اللغة الحالية مناسبة، لن يتم أي تبديل.

## الأوامر

- `autolanguage.toggle`: تشغيل/إيقاف مبدّل اللغة التلقائي.

## الإعدادات

| الإعداد | النوع | الافتراضي | الوصف |
|---|---|---|---|
| `autolanguage.enabled` | boolean | `true` | تمكين/تعطيل الامتداد |
| `autolanguage.showNotifications` | boolean | `true` | إظهار إشعار عند التبديل |
| `autolanguage.showStatusBar` | boolean | `true` | إظهار حالة الامتداد في شريط الحالة |

## إعدادات النظام حسب المنصة

### macOS

1. افتح إعدادات النظام.
2. ادخل إلى `Keyboard` ثم `Input Sources`.
3. أضف العربية والإنجليزية.

### Windows

1. افتح `Settings`.
2. ادخل إلى `Time & Language` ثم `Typing`.
3. من إعدادات لوحة المفاتيح المتقدمة أضف التخطيطات المطلوبة.

### Linux

تأكد من توفر أداة واحدة على الأقل حسب بيئتك:

- `gsettings` (GNOME)
- `setxkbmap` (X11)
- `ibus`
- `fcitx` أو `fcitx5`

## استكشاف الأخطاء

### رسالة أن تخطيط اللغة غير موجود

أضف تخطيطات العربية والإنجليزية من إعدادات النظام، ثم أعد محاولة الكتابة داخل VS Code.

### الامتداد لا يعمل

1. تأكد أن `autolanguage.enabled = true`.
2. تأكد من وجود التخطيطات في النظام.
3. راقب رسائل الخطأ داخل VS Code.

### على Linux: أدوات الإدخال غير مثبتة

```bash
# GNOME
sudo apt install gsettings-desktop-schemas

# X11
sudo apt install x11-xkb-utils

# IBus
sudo apt install ibus

# Fcitx
sudo apt install fcitx
```
